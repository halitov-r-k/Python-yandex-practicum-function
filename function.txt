Function

Блоки надо отделять друг от друга, и для этого в Python используются отступы.

Функция — это отдельный блок кода.


function declaration

Функцию объявляют ключевым словом def, 
затем указывают имя функции,
после имени - круглые скобки и двоеточие.
За объявлением функции следует код(тело функции), который функция должна выполнить при вызове. 
Тело функции, как и тело цикла, отбивается четырьмя пробелами от начала строки:
по этим отступам Python понимает, где начинается и заканчивается тело функции.
Нет отступов - нет функции. 

def function_name():
    #function_body
	
function call

Чтобы вызвать функцию - нужно написать название функции и  после названия поставить круглые скобки.	

У функции может быть и несколько параметров.
Способ передачи аргументов может быть позиционным или именованным. 
При вызове функции, при позиционом способе предачи аргументов, аргументы передаются в соответствии с порядком записи:
первый аргумент передаётся в первый параметр,
второй аргумент - во второй параметр.

def function_name(parametr1, parametr2):
    #function_body
	print(parametr1, parametr2)

function_name(argument1, argument2)	
	
Параметру может быть передано значение по умолчанию.
Это значение применяется если в функцию не передан аргумент.

def function_name(parametr1=argument1, parametr2=argument2):
    #function_body
	print(parametr1, parametr2)

function_name()	

При именованном способе передачи аргументов, в Функцию, при вызове, передается именованный параметр.

def function_name(parametr1=argument1, parametr2=argument2):
    #function_body
	print(parametr1, parametr2)
	
function_name(parametr2=argument3)

	
Переменную, при помощи которой ведут подсчёт чего-либо, принято называть count
Добавление одного и того же значения к переменной (в цикле или в других случаях) 
— достаточно частая операция, и для неё придумали специальный оператор: +=.

count = count+ value

заменяется надо

count += value

также существуют операторы: -=, *=, /-.

Сокращенная запись двойного неравенства

temp >= 22 and <= 26
заменить
22 <= temp <= 26

Функция может возврашать значение.Оно передается в ту часть кода, откуда функция была вызвана.
Возвращаемое значение указывается в теле функции после ключевого слова return.

def function_name(parametr1, parametr2):
    #function_body
	value = parametr1 operand parametr2
	return value
	
variable = function_name(argument1, argument2)

Возможен вызов функции из функции. 
При этом может возникнуть рекурсия.
Это может вызвать ошибку.
python обрабатывает такие ситуации, генерируя ошибку RecursionError.
